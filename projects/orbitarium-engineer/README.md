# Orbitarium Engineer Project

## Objective
AI engineering assistant specifically designed to aid development of the Orbitarium project through code review, architecture analysis, improvement suggestions, and implementation planning.

## Potential Integrations
- **Filesystem MCP** — Direct access to Orbitarium source files for examination and modifications
- **GitHub MCP** — Repository management, issue tracking, pull request collaboration workflow automation
- **Documentation MCP** — Technical documentation generation, maintenance, and knowledge base curation

## Key Tasks This Agent Performs
1. Review code quality identifying issues, improvements, and optimizations
2. Analyze architecture for scalability, maintainability, and best practices alignment
3. Suggest concrete implementation improvements with rationale
4. Plan feature implementations considering existing project constraints

## Architecture Components
- **Code Review Agent** — Quality analysis, best practices compliance checking
- **Architecture Analyzer** — System design evaluation and improvement suggestions
- **Documentation Generator** — Technical documentation creation and maintenance
- **Implementation Planner** — Feature planning with dependency awareness

## Experiment Ideas
- Automated code review for Orbitarium pull requests
- Architecture health scoring across project modules
- Documentation completeness analysis and gap identification
