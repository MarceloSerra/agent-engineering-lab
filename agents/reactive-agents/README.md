# Reactive Agents

## What is it?
Reactive agents operate on simple sense-act loops without explicit planning — they observe current state and immediately respond with appropriate actions based on environmental conditions.

## Characteristics
- Immediate response to stimuli without deliberation delay
- Simple condition-action rule mapping for quick decisions  
- Fast execution suitable for real-time interaction requirements
- Limited lookahead capability focusing only on present moment states
