# Planning Agents

## What is it?
Planning agents decompose complex goals into sequential subtasks before execution — creating structured action plans that guide systematic problem-solving approaches toward desired outcomes effectively.

## Key Features
- Goal decomposition into manageable sequential task components logically  
- Dependency tracking ensuring proper execution ordering maintained correctly always
- Progress monitoring providing completion status updates regularly throughout processes
- Adaptation capability modifying plans when obstacles encountered unexpectedly arise suddenly
