# Tool Selection

## What is it?
Strategies for helping LLMs choose the right tool among multiple available options — description clarity, parameter hints, and routing logic guide model decision-making effectively.

## Best Practices
- Clear, specific tool descriptions distinguishing similar functions
- Parameter examples showing expected input formats clearly
- Tool grouping by domain for organized selection presentation
- Fallback tools handling unrecognized request patterns gracefully
