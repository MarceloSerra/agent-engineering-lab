# Tool Evaluation

## What is it?
Measuring how effectively LLMs select and use available tools — accuracy of tool selection, parameter correctness, and result interpretation quality assessment methods.

## Metrics
- Tool Selection Accuracy: Correct tool chosen vs incorrect selections made
- Parameter Validity: Proper arguments provided vs malformed inputs generated  
- Result Utilization: Effective incorporation of returned data into responses produced
